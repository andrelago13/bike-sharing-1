\hyperlink{class_posto_servico}{Posto\+Servico} -\/ Auxiliary program class. 

This class lets us manage everything about bikes. Here we can rent or return a bike, fix broken bikes and remove bikes or users. Therefore is class that help us have a clear vision of what bike-\/sharing really is
